# Chatbot Trial user guide

This guide is for trial users who receive the Windows installer.

## Install

1. Double-click the `Chatbot-Setup-0.1.0-x64.exe` installer.
2. If Windows SmartScreen appears, choose **More info**, confirm the file source, then choose **Run anyway**.
3. Follow the installer wizard and keep the default options unless your maintainer gave you a different install location.
4. When installation finishes, leave **Launch Chatbot Trial** selected if you want to start immediately.

## Start Chatbot

- Double-click **Chatbot Trial** from the desktop or Start menu.
- Wait for the browser window to open.
- If the browser does not open, use the tray icon and choose **Open**.

## Tray icon

After startup, Chatbot keeps a tray icon in the Windows notification area.

- **Open**: opens the Chatbot page again.
- **Status**: shows whether Chatbot is ready.
- **Restart**: restarts Chatbot if the page stops responding.
- **Save diagnostics**: creates a diagnostic package for your maintainer.
- **Exit**: closes Chatbot cleanly.

## Real sending default

Real sending is disabled by default in the trial build. Do not enable sending unless your maintainer explicitly asks you to do so for a controlled test.

## Save diagnostics

1. Right-click the tray icon.
2. Choose **Save diagnostics**.
3. Send the generated diagnostic package path to your maintainer.

## Exit

1. Right-click the tray icon.
2. Choose **Exit**.
3. Wait until the tray icon disappears before starting Chatbot again.

## Uninstall

1. Open Windows **Settings**.
2. Go to **Apps**.
3. Select **Chatbot Trial**.
4. Choose **Uninstall**.
5. The default uninstall flow keeps your user data. Only remove user data when your maintainer explicitly asks and you understand the result.
